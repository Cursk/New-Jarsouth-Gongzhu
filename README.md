# New-Jarsouth-Gongzhu 新日尔南拱猪
中国传统纸牌游戏“拱猪”线上版（新日尔南：新西兰+日本+哈尔滨+济南）。因为没有现成的小程序和app所以自己做了一个。在replit上编译并测试过，拿自己主机当服务器就可以噜。作者是电气专业，啥也不懂，有bug欢迎指出。谢谢朋友们！(*╹▽╹*)
致报春/蕾蕾/桌儿/晓宇哥.jpg

An online version of the traditional Chinese card game ‘Gong Zhu’. As no existing mini-program or app was available, I developed one myself. It has been compiled and tested on Replit, pls use your own host as the server.

I am an electrical engineer with no expertise in this field so please feel free to point out any bugs. Thanks (*╹▽╹*)
